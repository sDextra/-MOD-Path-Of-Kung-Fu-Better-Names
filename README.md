"PATH OF KUNG FU"
This is a small quality-of-life mod that replaces the default English/Western random names (John Smith, Mary Johnson, etc.) with proper Chinese-style names in pinyin.
Now newly spawned NPCs, family members and other characters will get names like Li Shan, Wang Mei, Chen Hao, Zhao Fei, Ma Xue, Yang Kai and so on — exactly the way it feels right in a wuxia world.

What this mod changes:
- All surnames > common Chinese surnames (Li, Wang, Sun, Chen, Ma, etc.);
- Male given names > typical masculine pinyin names;
- Female given names > typical feminine pinyin names.

Newly generated NPCs should now have pinyin names.
Important notes:
- Only for game version 1.1.0.0;
- Only affects new games.

Installation (manual):
1. Download the attached file: [i]WorldEvolution.xlsx[/i];
2. Go to your game folder:
Steam\steamapps\common\The road of Jianghu\daojianjianghulu_Data\StreamingAssets\resdata\ExcelData\;
3. Make a backup of the original [i]WorldEvolution.xlsx[/i] (just in case);
4. Replace the original file with the downloaded one;
5. Launch the game.
   
Uninstallation:
Just put the original (backup) WorldEvolution.xlsx back into the folder.

Enjoy a more authentic feel!
